Keeviqo Platform - Access Instructions

Local Access

The Keeviqo platform is currently running locally on your machine at:

http://localhost:3000

You can access it directly in your browser by clicking the button below:

Open Keeviqo Platform

  • Password: Keeviqo2023!

Remote Access

To access the platform from another device, you'll need to set up a tunnel. Here are two options:

Option 1: Using localtunnel

Run the following command in your terminal:

npx localtunnel --port 3000

Option 2: Using ngrok

First, sign up for a free account at ngrok.com

Then run these commands:

npm install -g ngrok
ngrok config add-authtoken YOUR_AUTH_TOKEN
ngrok http 3000
